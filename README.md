# Australian Open Data Analytics & Visualization

## Overview

This project analyzes historical Australian Open tennis data to identify player performance trends, championship statistics, and country-level insights using data visualization techniques.

## Objectives

- Explore historical Australian Open data
- Analyze player and tournament performance
- Identify long-term trends and patterns
- Create interactive dashboards for decision-making

## Technologies

- Tableau
- Data Analytics
- Data Visualization
- Exploratory Data Analysis (EDA)

## Features

- Interactive dashboards
- Player performance analysis
- Championship statistics
- Country-level insights
- Trend analysis

## Key Skills

- Data Cleaning
- Data Transformation
- Exploratory Data Analysis
- Dashboard Development
- Data Storytelling
